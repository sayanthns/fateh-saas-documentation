Items

Items are the products and services you buy and sell. They are the building blocks of all invoices, orders, and stock entries.

Creating an Item

1

Go to Stock → Items → New

2

Fill in the basics

Enter Item Code (unique identifier), Item Name (display name), and Item Group (category).

3

Set stock and UOM

If this is a physical product, enable Maintain Stock. Set the Unit of Measure (Nos, Kg, Box, etc.).

4

Set pricing

Go to the Sales Price List tab and add the default selling price. Add a buying price in Purchase Price List.

5

Set tax template (optional)

In the Sales Details tab, set Item Tax Template to Saudi VAT 15% if this item is taxable.

Stock Items vs. Non-Stock Items

TypeDescriptionExample
Stock ItemPhysical goods tracked in warehousesProduct, spare part
Non-Stock ItemServices or consumables not trackedConsulting fees, delivery

Item Variants

For items that come in different sizes or colors (e.g., a T-shirt in Small, Medium, Large), enable Has Variants on the Item Template and define the attributes. Fateh will create separate variant items automatically.
