Navigation

Learn how to move around Fateh ERP — the workspace, modules, search, and shortcuts.

Home Workspace

After logging in, you land on the Home workspace. It contains shortcut cards for every module:

  • Selling — Customers, Quotations, Sales Orders, Sales Invoices
  • Buying — Suppliers, Purchase Orders, Purchase Invoices
  • Accounting — Journal Entries, Payment Entries, Reports
  • Stock — Items, Warehouses, Stock Entries

Click any card to open that module's workspace.

The Top Navigation Bar

ElementWhat it does
🔍 Search barSearch for any document, setting, or menu item by name
☰ Navbar menuAccess all modules and recent documents
🔔 NotificationsSee alerts, approvals, and system messages
👤 User avatarProfile, settings, and logout

Keyboard Shortcuts

ShortcutAction
Ctrl + /Open global search
Ctrl + SSave current document
Alt + ←Go back
Ctrl + NCreate new document (on list views)

Breadcrumbs

At the top of every page you'll see a breadcrumb trail like: Selling → Sales Invoice → INV-0001. Click any item in the trail to go back to that level.

Filters & Search in Lists

On any list view (e.g., Customer list, Invoice list), you can:

  • Type in the search box to filter by name
  • Click the filter icon (funnel) to add advanced filters (e.g., filter invoices by date range or status)
  • Click column headers to sort
